El desarrollo de sitios WEB

Se basa en la necesidad de empresas de tener presencia en el internet, ya que permite tener acceso a un mercado más amplio e internacional, aprovechando que es el nuevo medio de comunicación masivo más utilizado en la actualidad permitiendo expandir los alcances de las empresas. Un sitio web puede dar múltiples beneficios: puede ser un excelente medio de venta, una plataforma administrativa, o un sistema de control y monitoreo, depende de las necesidades de la empresa. Además, un sitio web permite que mayor cantidad de usuarios encuentren la información de manera fácil y rápida, dividiendo la página en prácticas secciones como chats, foros, correos electrónicos, galerías de imágenes, sistema de noticias, registro de usuarios, entre otros.

El desarrollo de sitios web permite a las empresas responder mejor a necesidades y ser adaptable a las necesidades futuras mediante la inclusión de módulos adicionales. Considerando el sitio como una plataforma tecnológica que organiza y administra la página de manera centralizada, se puede modificar de muchas maneras siendo sencillo de utilizar en ventas para ofrecer un catalogo de productos, con un sitio que cuente además con un carrito de compras online, donde permite llevar un inventario ordenado y visualizar las tendencias del mercado para tomar decisiones a futuro.

E-commerce

Conocido como comercio electrónico, es un método de comprar vender bienes y servicios mediante internet o de manera online. Esta modalidad de comercio se ha vuelto muy popular con el auge de Internet y la banda ancha, así como por el creciente interés de los usuarios a comprar por Internet.

El e-commerce cuenta con ventajas tales como: Permite una disponibilidad 24 horas durante los 365 días del año para el cliente. - No tiene limitaciones geográficas para el cliente. - Posibilidad de segmentar a los clientes al trabajar online, mejorando la comunicación y lanzando campañas especializadas. - Extender el alcance de tu negocio a nuevos usuarios.

Desarrollo de aplicaciones multiplataforma

Una aplicación o APP es un programa informático enfocado y diseñado para su utilización en dispositivos móviles. Las aplicaciones han caracterizan por tener una calidad superior a las webs tradicionales en diseño, capacidad, rendimiento y funcionalidades. Las aplicaciones multiplataforma tienen, además, una gran ventaja añadida: se pueden utilizar en cualquier dispositivo, independientemente del sistema operativo.

Funcionan en los principales sistemas operativos de los smartphones y tablets. - Gran ahorro de tiempo y costes económicos. - Posibilidad de desarrollar aplicaciones con la misma calidad y prestaciones que las nativas. - No necesita un navegador para hacer su descarga, esta se realiza de manera directa. - Máxima integración con el hardware y el software específicos de cada dispositivo.

Lenjuages de programacion WEB

El más conocido de los lenguajes de programación web es el HTML (Hiper Text Markup Language).Es el lenguaje usado para crear páginas web en Internet, permite codificar un documento y junto con el texto incluye unas etiquetas o marcas que le aportan información adicional sobre la forma y presentación de ese texto.

El HTML es no de los lenguajes mas usados ya que la mayoría de los navegadores de Internet lo toleran bastante bien, es uno de los lenguajes más usados para la creación de documentos y es un lenguaje muy fácil de aprender.

Existen otros lenguajes de programación web que también se usan para acompañar o mejorar la presentación de la página web, tales como CSS, hojas de estilo que mejoran y personalizan la presentación de la página, JavaScript, que permite dar efectos mucho más dinámicos al sitio. PHP, es el más conocido y usado de los lenguajes de programación web de servidor. ASP y JSP, son los más usados actualmente en la programación web y se pueden acoplar para trabajar conjuntamente con MySQL (lenguaje de programación para manejar bases de datos).

Responsive Web design

El diseño web receptivo es el enfoque que sugiere que el diseño y el desarrollo deben responder al comportamiento y al entorno del usuario según el tamaño de la pantalla, la plataforma y la orientación.

La práctica consiste en una combinación de cuadrículas y diseños flexibles, imágenes y un uso inteligente de las consultas de medios de CSS. A medida que el usuario cambia de su computadora portátil a iPad, el sitio web debe cambiar automáticamente para adaptarse a la resolución, el tamaño de la imagen y las capacidades. En otras palabras, el sitio web debe tener la tecnología para responder automáticamente a las preferencias del usuario. Esto eliminaría la necesidad de una fase de diseño y desarrollo diferente para cada nuevo dispositivo en el mercado

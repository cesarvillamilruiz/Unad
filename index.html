Mision

Que el OVI cumpla con sus expectativas a nivel de Diseño y Funcionalidad dando cumplimiento a los requerimientos segun la guia y el curso, y aparte de todo y no menos importante que adquiramos todo este conocimiento del Desarrollo Web.

Vision

Que al terminar este Curso tengamos el conocimiento de como Desarrollar un Sitio Web de forma manuscrita (Codigo linea a linea) para que en el futuro nos sirva como medio para generar ingresos.

Objetivo del OVI

El Objeto virtual desarrollado en el curso de Diseño de Sitios Web es una gran herramienta para el estudiante donde podra identficar cada una de las tematicas del curso como lecturas, videos, y autoevaluaciones en linea.

Objetivo del Curso

En el curso de diseño de Sitios Web es fundamental para los estudiantes de Ingenieria en Sistemas para apropiar conocimiento en el entorno de desarrollo de Sitios Web que de respuesta a Requerimientos a la nueva era de las Tic´s
